## 🧑‍💻 About Me

I’m **Kavinda Rathnayake**, a **Full-Stack Web3 Developer** passionate about building decentralized applications that redefine how we interact with technology.  

My journey started in **Web2 development** — working with **TypeScript, React, Next.js, Node.js, and PostgreSQL** to create scalable, performant, and user-friendly applications. Over time, that evolved into a deep passion for **blockchain technology**, where I now focus on **smart contract engineering** and **Web3 infrastructure**.  

I specialize in **Solidity** and the **Foundry** framework, developing secure and production-ready **dApps**, **DeFi protocols**, and **Web3 applications** on the **Ethereum blockchain**. I’ve also completed the full **[Cyfrin Updraft](https://www.cyfrin.io/updraft)** course series — mastering Solidity, EVM internals, and advanced Ethereum development.  

Currently, I’m learning **Rust** and **Solana program development**, exploring how performance-oriented blockchain ecosystems complement Ethereum’s security and decentralization.  

You can explore both my **Web2** and **Web3 projects** in my portfolio, each built with attention to detail, clean architecture, and a focus on scalability, security, and user experience.
